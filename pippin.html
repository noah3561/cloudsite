Pippin

The best new moderation bot! Pippin has utility commands such as serverinfo and userinfo, while also being highly customizable by being able to have per server prefixes, custom suggestion and report channels per server, and more!

Base Commands
Command Aliases Description Example
ping ping Returns the websocket and message ping of the bot p!ping
help help Returns the list of commands available to you p!help
avatar avatar Returns the requested users avatar, or the command authors avatar is no user is issued p!avatar (user)
invite invite Returns the invite for the bot p!invite
serverinfo serverinfo, server
si, guildinfo
guild, gi		 Returns information for the server you are currently in p!serverinfo
userinfo userinfo, user
ui		 Returns information for the user you requested p!userinfo (user)
shard shard Returns the shard that your server is currently on p!shard
uptime uptime Returns the bots uptime p!uptime
botstats botstats Returns the bots stats and module information p!botstats
suggest suggest Puts your suggestion into your guild's suggestion channel, if the suggestion system module is enabled and the suggestion channel is set p!suggest <suggestion>
report report Reports the specified user for the specified reason into your guild's suggestion channel, if the report module is enabled and the reports channel is set p!report <user> <reason>

Setting Commands
Command Aliases Description Example
settings settings Returns the current settings list for your guild p!settings
prefix prefix Used to set the custom prefix for your guild or to reset it back to p! p!prefix <set> <new prefix>
p!prefix <reset>
modlog modlog, mod-log Used to set the mod log channel for your guild or to reset it back to none p!modlog <set> <channel>
p!modlog <reset>
modrole modrole, mod-role Used to set the mod role for your guild or to reset it back to none p!modrole <set> <channel>
p!modrole <reset>
serverlog serverlog, server-log Used to set the server log channel for your guild or reset it back to none p!serverlog <set> <channel>
p!serverlog <reset>
memberlog memberlog, member-log Used to set the member log channel for your guild or reset it back to none p!memberlog <set> <channel>
p!memberlog <reset>
suggestsystem suggestsystem
suggest-system		 Used to enable or disable the suggestion system for your guild. p!suggestsystem <on/off>
suggestchannel suggestchannel
suggest-channel		 Used to set the suggestion channel for your guild to be used with the suggest command p!suggestchannel <set> <channel>
p!suggestchannel <reset>
reportsystem reportsystem
report-system		 Used to enable or disable the report system for your guild. p!reportsystem <on/off>
reportchannel reportchannel
report-channel		 Used to set the report channel for your guild to be used with the report command p!reportchannel <set> <channel>
p!reportchannel <reset>
